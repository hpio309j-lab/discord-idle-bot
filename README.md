# Discord Idle Bot

هذا المشروع عبارة عن بوت Discord مبني باستخدام Node.js يهدف إلى إبقاء حسابك في غرفة صوتية محددة بشكل دائم. يتضمن المشروع لوحة تحكم بسيطة عبر الويب لإدارة البوت وتشغيله/إيقافه.

## الميزات

- **تثبيت في الغرفة الصوتية:** يحافظ على وجود حساب Discord الخاص بك في غرفة صوتية محددة.
- **إعادة الاتصال التلقائي:** في حال تم فصل البوت من الغرفة الصوتية، سيحاول إعادة الاتصال تلقائيًا بعد 10 ثوانٍ.
- **كتم الصوت تلقائيًا:** يقوم البوت بكتم صوت نفسه تلقائيًا عند الانضمام إلى الغرفة.
- **لوحة تحكم ويب:** واجهة بسيطة عبر المتصفح لتشغيل/إيقاف البوت، إدخال التوكن ومعرف الغرفة، وعرض السجلات.
- **جاهز للاستضافة على Railway:** تم تصميم المشروع ليكون سهل النشر على منصة Railway.

## البدء السريع

### المتطلبات

- حساب Discord Bot Token (يمكنك الحصول عليه من [Discord Developer Portal](https://discord.com/developers/applications))
- معرف الغرفة الصوتية (Voice Channel ID) التي تريد أن يبقى البوت فيها.
- Node.js (v14 أو أحدث) و npm مثبتان محليًا إذا كنت ترغب في التشغيل المحلي.

### الإعداد المحلي

1.  **استنساخ المستودع:**
    ```bash
    git clone <رابط المستودع الخاص بك>
    cd discord-idle-bot
    ```

2.  **تثبيت التبعيات:**
    ```bash
    npm install
    ```

3.  **التشغيل:**
    ```bash
    npm start
    ```
    سيتم تشغيل خادم الويب على المنفذ 3000 افتراضيًا. افتح متصفحك على `http://localhost:3000` للوصول إلى لوحة التحكم.

### النشر على Railway

1.  **إنشاء مستودع GitHub:**
    تأكد من أن هذا المشروع موجود في مستودع GitHub خاص بك.

2.  **إنشاء مشروع جديد على Railway:**
    - اذهب إلى [Railway.app](https://railway.app/) وسجل الدخول.
    - انقر على `New Project` ثم `Deploy from GitHub Repo`.
    - اختر المستودع الخاص بهذا المشروع.

3.  **إعداد المتغيرات البيئية (Environment Variables):**
    بعد نشر المشروع، اذهب إلى إعدادات المشروع في Railway وأضف المتغيرات البيئية التالية:
    - `DISCORD_TOKEN`: توكن البوت الخاص بك من Discord.
    - `VOICE_CHANNEL_ID`: معرف الغرفة الصوتية التي تريد أن يبقى البوت فيها.
    - `PORT`: (اختياري) المنفذ الذي سيعمل عليه التطبيق، Railway ستقوم بتعيينه تلقائيًا.

4.  **النشر:**
    سيقوم Railway تلقائيًا ببناء ونشر التطبيق. بمجرد الانتهاء، ستحصل على عنوان URL عام للوصول إلى لوحة التحكم.

## كيفية الحصول على Discord Bot Token ومعرف الغرفة الصوتية

### Discord Bot Token

1.  اذهب إلى [Discord Developer Portal](https://discord.com/developers/applications).
2.  سجل الدخول بحسابك في Discord.
3.  انقر على `New Application` وأعطِ تطبيقك اسمًا.
4.  في الشريط الجانبي الأيسر، اختر `Bot`.
5.  انقر على `Add Bot` ثم `Yes, do it!`.
6.  تحت قسم `TOKEN`، انقر على `Reset Token` ثم `Copy` لنسخ التوكن الخاص بك. **لا تشارك هذا التوكن مع أي شخص!**
7.  تأكد من تمكين `PRESENCE INTENT` و `SERVER MEMBERS INTENT` و `MESSAGE CONTENT INTENT` في قسم `Privileged Gateway Intents`.
8.  لإضافة البوت إلى خادمك، اذهب إلى `OAuth2` -> `URL Generator`.
9.  في قسم `SCOPES`، اختر `bot`.
10. في قسم `BOT PERMISSIONS`، اختر الصلاحيات اللازمة مثل `Connect`, `Speak`, `Mute Members`.
11. انسخ الرابط الذي تم إنشاؤه والصقه في متصفحك لإضافة البوت إلى خادمك.

### معرف الغرفة الصوتية (Voice Channel ID)

1.  افتح Discord على جهاز الكمبيوتر الخاص بك.
2.  اذهب إلى `User Settings` -> `App Settings` -> `Advanced`.
3.  قم بتمكين `Developer Mode`.
4.  انقر بزر الماوس الأيمن على الغرفة الصوتية التي تريد استخدامها، ثم اختر `Copy ID`.

## ملاحظات

- هذا البوت مصمم للبقاء في غرفة صوتية واحدة فقط.
- لا يقوم البوت بالدفاع عن نفسه (defend) أو إلغاء كتم الصوت (unmute)؛ يظل صامتًا ومكتومًا.
